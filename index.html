What file types does SFMT support?

+

SFMT supports a wide range of file types including documents (PDF, DOC, DOCX, TXT), images (JPG, PNG, GIF) and many more. While our semantic analysis capabilities are influenced by the depth of the file's location, the results are ultimately determined by the file query submitted to the tool.

What are the system requirements?

+

SFMT runs on Windows 10/11 (64-bit). We recommend at least 4GB of RAM and 500MB of free disk space. For optimal performance when managing large file collections, 8GB+ RAM is recommended.
